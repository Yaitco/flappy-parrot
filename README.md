# flappy-parrot
Реализация игры flappy bird в сетинге попугаев

## Инструкция к установке
1. Клонирование репозитория:
   ```bash
   git clone https://github.com/Yaitco/flappy-parrot
   cd flappy-parrot
2. Запуск setup:
   ```bash
   ./setup.sh
3. Запуск игры, наслаждайтесь:
   ```bash
   flappy-parrot
